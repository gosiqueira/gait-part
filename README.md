# GaitPart: Temporal Part-based Model for Gait Recognition with TensorFlow 2.0

## Description

[TensorFlow 2.0](https://www.tensorflow.org/) implementation of `GaitPart` published on CVPR 2020.

## Requirements

* TensorFlow 2.x
* TensorBoard

## Contributions
